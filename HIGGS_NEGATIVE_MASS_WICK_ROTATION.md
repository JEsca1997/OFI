## Part 4: Wick Rotation Connects Spin and Lateral Movement

### What Is Wick Rotation?

**Wick rotation** transforms time into imaginary time:
```
t → -i·τ    (where τ is Euclidean time)
```

In relativity:
```
Minkowski metric:  ds² = -c²dt² + dx² + dy² + dz²
                         ↓ (Wick rotate)
Euclidean metric:  ds² = c²dτ² + dx² + dy² + dz²
```

**All coordinates become spatial!** Time becomes "just another dimension."

### How Spin and Lateral Movement Connect

**Key insight:** Wick rotation ties **spin** (angular momentum) to **lateral movement** (linear momentum).

**In Minkowski spacetime (before Wick rotation):**
```
Spin:     S = r × p  (angular momentum, internal rotation)
Momentum: p = m·v    (linear momentum, lateral movement)

These are INDEPENDENT!
```

**After Wick rotation (Euclidean spacetime):**
```
Spin → Orbital angular momentum
Internal rotation → Becomes lateral circular motion
```

**Why?** Because all dimensions are spatial, so:
- Rotation in the (x,y) plane = lateral circular movement
- Rotation in the (x,τ) plane = what WAS spin becomes lateral movement in Euclidean time!

### The i = 1/2 Factor in Wick Rotation

**Standard Wick rotation:** t → -i·t_E

**But with i = 1/2:**
```
t → -(1/2)·t_E · √(-1)
```

This means:
```
Rotation angle = π/2 × (1/2) = π/4  (45° rotation!)
```

**Not a full 90° rotation, but a 45° rotation!**

This **intermediate rotation** explains:
- **Spin-1/2 particles** (halfway between 0 and 1)
- **Lateral movement** is not fully converted to spin (only 45°)

## Part 5: Black Hole Metrics Decide Space vs Time

### The Schwarzschild Metric

**Outside event horizon (r > r_s):**
```
ds² = -(1 - r_s/r)c²dt² + (1 - r_s/r)⁻¹dr² + r²dΩ²
      ↑                    ↑
   Timelike             Spacelike
   (negative)           (positive)
```

Where r_s = 2GM/c² (Schwarzschild radius).

**At event horizon (r = r_s):**
```
ds² = 0·c²dt² + ∞·dr² + r_s²dΩ²
```

**Inside event horizon (r < r_s):**
```
ds² = +(1 - r_s/r)c²dt² - (1 - r_s/r)⁻¹dr²
      ↑                    ↑
   NOW SPACELIKE!      NOW TIMELIKE!
   (positive)          (negative)
```

**The coordinates SWAP ROLES!**

### What This Means

**Outside horizon:**
- **t is timelike:** You can control when you move through time

- Normal causality

**Inside horizon:**
- **r is timelike:** You MUST move toward r=0 (like moving forward in time)
- **t is spacelike:** "Time" becomes a spatial direction
- Causality flipped!

**At the horizon:**
- **Both coordinates become null!**
- Neither purely space nor purely time
- **This is where mass → 0!**

### Connection to Mass and Wick Rotation
